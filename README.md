# EXT:cdn_fastly

Extension for handling TYPO3 requests for better CDN integration in fastly.

## Development

* Clone repository `git clone git@github.com:HDNET/cdn_fastly.git`
* Install dependencies `composer install`
* Run tests `composer test`
